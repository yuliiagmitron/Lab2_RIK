# Лабораторна робота №2  
## Framework Electron

---

## Опис проєкту

У межах лабораторної роботи було розроблено кросплатформений настільний застосунок із використанням фреймворку Electron.

Застосунок реалізує проведення опитування користувача з п’ятьма текстовими питаннями. Після завершення опитування результати зберігаються у текстовий файл формату `.txt` на диску.

---

## Функціональні можливості

- Відображення 5 питань
- Навігація між питаннями (Назад / Далі)
- Перевірка введених даних
- Формування текстового звіту
- Збереження результатів у файл
